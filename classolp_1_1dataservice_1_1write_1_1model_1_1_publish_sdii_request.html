Detailed Description

PublishSdiiRequest used to send a list of SDII messages to a stream layer. SDII message data must be in SDII MessageList protobuf format. For more information please see the HERE platform Sensor Data Ingestion Interface documentation and schemas.

Note
the Content-Type for this request is always "application/x-protobuf".
SDII MessageList data must be encoded in protobuf format according to the HERE platform SDII Message List schema. The maximum size is 20 MB.
