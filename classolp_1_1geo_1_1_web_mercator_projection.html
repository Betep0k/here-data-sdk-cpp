Detailed Description

Web mercator projection.

Latitude between -MaximumLatitude() and MaximumLatitude() map to Y world axis with range: [0, EarthConstants::EquatorialCircumference()] Longitude between -180 and 180 map to X world axis with range: [0, EarthConstants::EquatorialCircumference()] Altitude maps to Z world axis and is not scaled.
